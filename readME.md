## dances.uAgent

### syntax
#### 获取浏览器客户端
	dances.uAgent.whois

#### 获取浏览器核心引擎
	dances.uAgent.engine

#### 获取浏览器版本号
	dances.uAgent.ver

#### 嗅探
	dances.uAgent.firefox
	dances.uAgent.chrome
	dances.uAgent.msie
	dances.uAgent.opera
	dances.uAgent.safari

	dances.uAgent.gecko
	dances.uAgent.trident
	dances.uAgent.webkit
	dances.uAgent.presto

## dances.addCss

### syntax
	dances.addCss(ss[, headEl]);

+ ss
css rule

+ headEl
可选, style element 添加到的元素
